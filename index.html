    使用教程

    🎨 自定义与个性化

    • 一键导入美化：支持导出/导入所有自定义美化设置
    • 自定义APP图标：设置 → APP图标 → 上传图片替换所有图标
    • 个性化气泡：每个聊天页面可设置独立CSS气泡样式（置空则使用全局气泡）
    • 切换显示模式：新增灵动岛显示效果切换（记得在设置里选择安卓/iOS模式）
    • 线下隐藏头像：通过独立CSS实现，例如：.chat-messages.offline-mode .message-wrapper .avatar { display: none !important; }

    💬 聊天功能

    • 消息操作：双击消息可进行重置、编辑（包括文字、语音、图片等所有格式），系统消息（灰色）不可编辑
    • 图片显示：长按图片显示文字（不会自动隐藏）
    • 消息管理：多选消息批量删除、一键清空聊天记录、搜索聊天记录
    • 引用消息：AI可正确引用对话内容进行回复
    • 线上/线下模式：分别支持独立CSS气泡，线下模式兼容HTML
    • 操作提示：消息接收时接收按钮会顺时针旋转，成功后停止；遇到问题可关闭页面重新刷新（不影响存档）

    😊 表情包功能

    • 批量导入表情包
    • 三行显示表情包，支持批量删除
    • 点击灵敏度优化（滑动不会误触发送）
    • 点击框外任意位置返回

    🎵 音乐功能

    • 搜索音乐：支持搜索功能（链接时效性短，建议使用链接导入）
    • 一起听功能：右上角点击"加入一起听"或在聊天页面左下角功能栏开启
    • 互动与控制：聊天框内可实时交流，按回车发送，右键返回查看AI回复；AI可控制切歌、暂停、下一首
    • 退出一起听：点击退出会将聊天记录保存到主聊天中
    • 动图播放器：音乐页面支持动图显示

    📝 日记与心声

    日记功能

    • 主动生成：使用主API
    • 自动触发：使用副API
    • 支持导入导出，可调节日记间隔时间（自定义日记生成频率）

    心声功能

    • 点击聊天顶部AI名字区域主动查看，接收完消息自动弹窗
    • 支持滚动查看，点击框外区域关闭
    • 右上角开关控制主动弹窗（✓=开启，✗=关闭）

    🔧 API设置

    • 双API预设：主API和副API可保存切换（建议副API使用免费/便宜的）
    • 主动搭话：频率可调
    • 尾部破限：可自行选择开启/关闭

    💰 钱包与商城

    • 转账功能：AI可以退钱
    • 钱包系统：打工挣钱或向AI乞讨
    • 商城功能：点击右上角设置按钮添加商品，点击管理按钮批量删除商品
    • 商品图标：支持emoji或自定义URL作为商品图标
    • 使用商品：点击商品 → 选择目标角色 → 发送礼品卡片给角色

    🎯 其他功能

    • 拍一拍：互动功能
    • 语音聊天：文字格式（无真实语音功能），支持滚动查看，单击语音条操作
    • 朋友圈：AI可正常回复互动
    • 随笔：新增关闭按钮
    • 全屏显示：支持iOS真全屏

    💡 遇到问题？尝试关闭页面重新刷新，或检查 API 配置
